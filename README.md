## 项目简介

TalktoArt 是一款运行在浏览器中的纯语音控制绘图应用。用户无需触碰鼠标或键盘，仅通过自然语言语音指令即可完成从简单图形到多对象组合的完整绘图创作。

> 说一声"画一个日出风景"，画布自动构建太阳、山、河流和树。

## 功能一览

### 🎤 语音控制
| 功能 | 说明 |
|------|------|
| 唤醒词激活 | 说 **"小A小A"** 唤醒，说 **"不画了"** 休眠 |
| 连续识别 | 说完一句自动等待下一句，无需反复点击 |
| 语音反馈 | 每个操作后 TTS 语音播报结果（"已画出黄色太阳"） |
| 自动启动 | 打开页面 mic 自动就绪 |

### 🎨 绘图能力
| 功能 | 示例指令 |
|------|---------|
| 基本形状 | "画一个红色的圆" / "画个蓝色方形" / "画个绿色三角形" |
| 自由绘制 | "开始画画" → 拖动鼠标涂鸦 → "完成" |
| 线条绘制 | "画一条蓝色线" |
| 语义对象 | "画一棵树" → 🟩 绿色三角形，"画个房子" → 🟫 棕色矩形 |
| 批量绘制 | "画三朵花" → 自动网格排列 |

> 语义对象已全面升级为**组合图形**：树（树干+树冠）、太阳（圆盘+光芒）、
> 河流（多层水纹）、花朵（花瓣+花蕊）、房子（墙体+屋顶+门）、
> 云朵（多圆堆叠）、星星（光点+闪耀效果）

### 🖌️ 编辑操作
| 功能 | 示例指令 |
|------|---------|
| 改色 | "改成红色" / "全部改成蓝色" |
| 缩放 | "放大" / "缩小" / "全部放大" |
| 移动 | "向左移" / "移到中间" / "移到底部" / "向上移" |
| 旋转 | "旋转45度" / "顺时针转90度" |
| 选中 | "选中太阳" / "选中红色圆形" / "全选" |
| 多选 | "选中中间的红色圆形和右上角的绿色方形" |
| 撤销/重做 | "撤销" / "重做" |
| 删除 | "删除这个" / "删除所有" |
| 清空 | "清空画布" → 语音确认后执行 |

### 🏞️ 场景模板
| 场景 | 说出口令 | 自动绘制 |
|------|---------|---------|
| 🌅 日出 | "画一个日出风景" | 太阳 + 山 + 河流 + 树 |
| 🏙️ 城市 | "画一个城市" | 多栋建筑 + 道路 |
| 😊 笑脸 | "画一个笑脸" | 圆形 + 眼睛 + 嘴巴 |
| 🌸 风景 | "画个风景画" / "山水画" | 太阳 + 山脉 + 河流 + 树 |
| ⭐ 星空 | "画个星空" | 深色背景 + 多颗星星 |

> 场景模板自动逐步执行，每步间隔 600ms，画布实时展示构建过程。

> 所有场景中的元素均已升级为**组合图形**——树、太阳、河流、花朵、
> 星星等都会以完整组合形态绘制。

### 📐 智能网格排列

| 功能 | 说明 |
|------|------|
| 自动避让 | 连续绘制多个图形时自动网格排列，每行最多 3 个，超出自动换行 |
| 保持居中 | 新增图形时已有图形自动左移，整体组始终保持在画布正中 |
| 统一重排 | 普通图形和组合图形统一参与网格重排，场景模板中的固定位置图形除外 |

### 🤖 AI 智能解析
| 功能 | 说明 |
|------|------|
| 自然语言理解 | 说"在中间画一个大的红色圆" → AI 自动拆解 shape+color+size+position |
| 语义兜底 | AI 解析失败时自动降级到规则引擎，不中断体验 |
| 上下文记忆 | AI 记住上一步操作，"把它变成蓝色" 能理解"它"指什么 |
| 跨句指代 | "画一棵树" → "把它变大" → 自动引用上一步画的树 |
| DeepSeek 驱动 | 可选配置 API Key 开启 AI 模式，不开也不影响基础功能 |

### 📦 其他
| 功能 | 说明 |
|------|------|
| 导出图片 | "导出" / "保存" → 二次确认（说"确认"/"确定"/"好"/"可以"等任意确认词即可，不等TTS说完） → 下载 PNG |
| 帮助 | "帮助" / "有什么指令" → TTS 播报指令列表 |
| 画布查询 | "画布上有几个对象" → TTS 播报统计 |
| 状态栏 | 实时显示语音状态、唤醒状态、画笔模式 |

## 快速开始

### 环境要求

- **浏览器**: Chrome 90+ / Edge 90+（Web Speech API 支持）
- **设备**: 需要麦克风权限
- **网络**: 首次加载后无需网络（AI 模式可选）

### 本地运行

```bash
git clone https://github.com/eluzekidano910-eng/talk-to-art.git
cd talk-to-art
npm install
npm run dev
```

打开浏览器访问 `http://localhost:5173` 即可使用。

### 生产构建

```bash
npm run build
npm run preview
```

## 使用指南

### 三步上手

```
1️⃣ 打开页面 → 麦克风自动启动
2️⃣ 说 "小A小A" → 唤醒语音助手
3️⃣ 说 "画一个红色的圆" → 画布出现红色圆形
```

### 完整演示流程（90 秒）

```
唤醒:    "小A小A"
绘图:    "画一个日出风景"        → 场景模板逐步构建
编辑:    "把太阳变成红色"        → 颜色编辑
         "全选"                 → 选中所有图形
         "全部放大"              → 批量缩放
导出:    "导出"                 → 下载 PNG
休眠:    "不画了"               → 进入待命
```

全程无需触摸鼠标键盘。

### 开启 AI 解析（可选）

1. 点击右上角 ⚙ 齿轮图标
2. 输入 DeepSeek API Key
3. 开启 AI 模式开关

AI 模式下支持更自然的语言表达：

```
"在中间画一个大的红色圆形，然后在它下面画一个蓝色的方形"
"把太阳移到右上角并变成橙色"
"在河旁边画三棵树"
```

## 技术架构

```
┌─────────────────────────────────────┐
│           Browser (SPA)              │
│  ┌─────────┐  ┌─────────┐  ┌─────┐  │
│  │ Voice    │  │ Command │  │Canvas│  │
│  │ Input    ├──► Parser  ├──►Engine│  │
│  │(WebSpeech)│  │(AI+Rule)│  │Fab. │  │
│  └─────────┘  └─────────┘  └─────┘  │
│  ┌─────────┐  ┌─────────┐  ┌─────┐  │
│  │ TTS     │  │ Scene   │  │ UI  │  │
│  │Feedback │  │Template │  │Comp.│  │
│  └─────────┘  └─────────┘  └─────┘  │
└─────────────────────────────────────┘
```

### 技术栈

| 领域 | 技术 | 选型理由 |
|------|------|---------|
| 框架 | React 18 + TypeScript | 类型安全，社区生态最强 |
| 构建 | Vite 6 | 秒级 HMR，零配置 |
| 画布 | Fabric.js 6 | 内置图形/画笔/编辑，减少 60%+ 画布代码 |
| 语音识别 | Web Speech API | 浏览器原生，免费零延迟 |
| 语音合成 | SpeechSynthesis API | 浏览器原生，无需 TTS 服务 |
| 语义解析 | DeepSeek API + 规则引擎 | AI 兜底双保险 |
| 样式 | Tailwind CSS | 原子化 CSS，快速构建 UI |
