# PR 3：指令解析器与语音-画布链路

## 概述

在语音识别（PR1）和画布引擎（PR2）之间加一层指令解析器，打通从"说话"到"画图"的完整链路。用户说出"画一个红色圆形"，经过解析后直接在画布上绘制。

## 依赖

- PR1（语音识别）— 提供语音文本输入
- PR2（画布引擎）— 提供绘图执行能力

本分支从 `feature/canvas-engine` 切出，包含 PR2 的全部 3 个 commit。

## 计划提交（6 个）

## 验收标准

- [ ] 说出"画一个红色圆形" → 画布出现红色圆
- [ ] 说出"画蓝色矩形在右下角" → 蓝色矩形在右下角
- [ ] 说出"画一个很大的绿色三角形" → 大尺寸绿色三角形
- [ ] 说出"画三条线" → 三条线
- [ ] 说出"画一个黄色太阳，再画三座绿色的山在下面，最后加一条蓝色的河流" → 一句话拆成多个动作依次执行
- [ ] 命令日志面板逐条展示解析过程
- [ ] 至少支持 10 条自然语言绘图指令

## 架构示意

```
用户语音
  → VoiceRecognizer（PR1）
  → 文字 "画一个红色圆形"
  → CommandParser（本 PR）
     → 标准化 → 意图识别 → 参数抽取 → 缺省补全
  → { intent: 'draw', shape: 'circle', color: 'red', position: 'center' }
  → CanvasEngine.drawCircle()（PR2）
  → 画布更新 + CommandLog 记录
```
