# PR 5：命令日志增强 + 音效反馈

## 概述

在 PR4 完成编辑闭环后，PR5 补齐**听觉反馈**和**日志可视化**两个感知层。用户说话后不仅能看到画布变化和日志文字，还能听到对应的音效提示（成功「叮」、失败「嘟」），形成"说→做→听→看"的完整闭环。同时 CommandLog 面板增加执行耗时、展开详情等增强交互。

## 依赖

- PR1（语音识别）— 提供语音文本输入
- PR2（画布引擎）— 提供绘图执行能力
- PR3（指令解析器 + 词典）— 提供指令解析
- PR4（撤销/编辑/删除/导出）— 提供编辑闭环能力

## 验收标准

- [ ] 说"画一个红色的圆"→ 画圆 + 日志记录 + 听到「叮」
- [ ] 说无意义指令 → 日志显示错误 + 听到「嘟嘟」
- [ ] 无目标说"放大"（edit 失败）→ 播放「嘟」
- [ ] 日志面板可见执行耗时（ms）
- [ ] 单条日志可展开/收起查看解析过程
- [ ] 连续快速说指令时，前一条音效被后一条打断
- [ ] 日志条目接近 50 条时 UI 弱提示

## 架构示意

`
用户语音
  → VoiceRecognizer（PR1）   → 文字 "画一个红色的圆"
  → CommandParser（PR3）      → { intent: 'draw', shape: 'circle', color: 'red' }
  → CanvasEngine（PR2 + PR4） → 画布绘制 + HistoryManager.saveState()
  → SoundPlayer（本 PR）      → play('success') → 「叮」
  → CommandLog（PR3 + 本 PR） → 新增日志条目（带耗时 + 可展开）
  → SpeechBubble + MicButton  → 状态回 idle，等待下一条指令
`
